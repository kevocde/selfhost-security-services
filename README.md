# Welcome to Home Security
This is a project with multiple dockerized applications, to improve the security in your home.

## The services
### 1. Vault Warden
Vault Warden is a re implementation of the BitWarden server, compatible with all its extentions and integrations.

### 2. Wire Guard
Wire Guard is a VPN Open Source and good performance, and provides a way to connect always from a secure origin.

### 3. PI Hole
PiHole too Open Source, is a DNS server to block custom lists of domains that send sensible information, more that ads blocker is a good tool for your security.
